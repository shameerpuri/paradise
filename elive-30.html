It was eight years until the developers of the Linux Elive distribution were able to launch a new version . Elive is a very stable , non-commercial free operating systemthat may be a good option to replace your default system on your computer.

This system is maintained by a donation system and is developed by a single person.Elive provides an intuitive experience in a system completely ready for daily use.

 

 

 

 

The desktop is a very personalized version of Enlightenment that offers a light and beautiful experience that works great even on older computers.

This Linux distribution is based on the GNU / Linux Debian distribution which is based on providing a perfectly stable and powerful system.

It can be run directly in live mode without the need to install with its own guided persistence feature.

 

 

 

In addition to this if you want to turn your computer into a powerful workstation, this Linux distro has a very friendly installer that includes many features to facilitate the process.

Elive is a mix between an easy-to-use system for inexperienced users and includes incredibly useful tools for the advanced , this hybrid system so to speak comes in the form of a clean and beautiful desktop but powerful for any task.

Of the main features that we can highlight from this distribution are:

  • Lightness and speed
  • Effects and a beautiful desktop
  • Intuitive and simple
  • Guided and automated
  • Full, full of applications and features
  • Multimedia
  • Own features and integration
  • Futuristic and clean desk
  • Comfortable to work
  • It is not necessary to install and save sessions
  • Installer with migration, update and automatic modes
  • It works with 256 MB of RAM and 500 MHz CPU
  • Full root access
  • Programming functions

About the new release of Elive

A few days ago the developer of the Linux distribution Elive announced through a statement on his blog the availability of the new version of the distribution.

Thus Thanatermesis blog announcement:

After 8 years of silent development, the third stable version of Elive is available, the result is simply incredible and the integration is magnificent, it is not even possible to describe each internal function and the new website only contains a small portion of its features.

Unfortunately, not everything is rainbow and perfection, the lack of resources caused the launch to be delayed too much, and this led to old packages and controllers, but despite that, the final result is really worth it, Elive 3.0 is the most useful that was created.

Under the hood, Elive 3.0 is based on Debian 7 (“Wheezy”), along with the adapted Enlightenment 17 desktop environment .

Elive only requires 192 MB of main memory according to system requirements and is only available as a 32-bit version , with kernel that uses PAE. The distribution is not compatible with UEFI firmware, but it also runs on older hardware.

The latter, however, takes 24 hours.

Basically in this new version of the distribution we can highlight:

  • Improved designs and beautiful interfaces made by professional designers.
  • Better performance, integrity and features.
  • More tools to make your own hacks and customizations.
  • Make many ideas come true.
